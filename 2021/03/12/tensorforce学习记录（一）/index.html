tensorforce学习记录之DeepQNetwork

最近研究强化学习的时候使用了tensorforce中的$DeepQNetwork$框架，目前对我来说它还像一个黑箱。我提供状态参数给它，它反馈给我动作。这篇博客的目的是记录下学习tensorforce官方文档的过程。

参数部分

states

状态规范（必需，最好是通过environment参数中的Agent.create(...)隐式指定），任意嵌套描述状态的字典（通常是从Environment.states()获取），要求包含以下属性：

  • type (“bool” | “int” | “float”) - 状态数据格式（默认为:”float”）
  • shape (int | iter[int]) – 状态参数规格（必需）
  • num_values (int > 0) – 离散状态价值的数量 (格式必须为 “int”)
  • min_value/max_value(float) – 最小/最大状态价值 (optional for type “float”).

    actions

    状态规范（必需，最好是通过environment参数中的Agent.create(...)隐式指定），任意嵌套描述动作的字典（通常是从Environment.actions()获取），要求包含以下属性：
  • type (“bool” | “int” | “float”) - 动作数据格式（默认为:”float”）
  • shape (int | iter[int]) – 动作参数规格（必需）
  • num_values (int > 0) – 离散动作价值的数量 (格式必须为 “int”)
  • min_value/max_value(float) – 最小/最大动作价值 (optional for type “float”).

    max_episode_timesteps（int>0）

    每次循环时间步数的上限

    memory (int > 0)

    循环使用的内存能力（必需）

    batch_size

    每一批更新的时间步数（必需）

目前的困扰是如何较好地设置state，并且state是如何影响到action？
